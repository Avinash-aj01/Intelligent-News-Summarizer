# Intelligent-News-Summarizer
The application integrates the language service, the translator API and the speech services of Microsoft Azure to summarise, translate and vocalise the news article.
The UI enables the user to paste an article as input and summarises the news article which is abstractive in nature. Moreover, it facilitates translation of the summary into any language chosen by the user. 
Finally, it produces a clear speech of the translation.
The integration of the three AI services of Microsoft Azure makes the application intelligent and versatile.
